## Project Summary

Musical Statues Web is a browser-based, facilitator-led game that uses Spotify playback and a Windows Media Player–style reactive visualisation layer.

A host signs in with Spotify, selects a playlist, starts a round, and controls or automates music stop/start moments while players follow the classic Musical Statues rules.

## MVP Outcome

Deliver a first version that is fun, legible, and technically robust enough for a host to run a game session from a laptop connected to speakers or a shared display.
